# Project Overview

## Introduction

This project performs exploratory data analysis on cryptocurrency market data using Python.

The goal is to understand:

- Market trends
- Price fluctuations
- Trading volume behavior
- Correlation between cryptocurrencies

## Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insight Generation

## Key Insights

- Bitcoin dominates the cryptocurrency market
- Ethereum price movement is strongly correlated with Bitcoin
- Altcoins show higher volatility compared to major coins

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab
