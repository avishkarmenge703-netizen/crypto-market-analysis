# Cryptocurrency Market Analysis using Python

## Project Overview
This project analyzes cryptocurrency market trends using Python. 
The goal is to explore price movements, trading volumes, and relationships between major cryptocurrencies.

## Objectives
- Analyze historical cryptocurrency price data
- Identify market trends
- Visualize price movements
- Analyze correlations between cryptocurrencies

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / Google Colab

## Dataset
The dataset contains cryptocurrency market data including:

- Date
- Cryptocurrency Name
